Tag <code>

Está tag ela define um pedaço de código de computador. Ex. Crie uma variavél com o nome: meuexemplo

Tag <dfn>

Está tag representa a instância de definição de um termo em HTML. Ex. Comando que especifica uma seção: blockquote

Tag <pre>

Está tag define o texto pré-formatado. Texto em um elemento pre é exibido em uma fonte de largura fixa (geralmente Courier) e preserva espaços e quebras de linhas.

Tag <span>

Está tag é usada para agrupar e aplicar estilos a elementos inline. A diferença entre a tag <span> e a tag <div> é que ela é usada com elementos inline enquanto a tag <div> é usada como conteúdo em nivel de bloco. Ex. Este texto está escrito dentro da tag <span>

Tag <time>

Está tag define uma data/hora legível por humanos. Esse elemento também pode ser usado para codificar datas e horários de forma legível por máquina, de modo que os agentes de usuário possam oferecer para adicionar lembretes de aniversário ou eventos agendados ao calendário do usuário e os mecanismos de pesquisa podem produzir resultados de pesquisa mais inteligentes.
